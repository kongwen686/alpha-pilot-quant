# Code of Conduct

AlphaPilot Quant follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive.
- Giving actionable technical feedback.
- Accepting responsibility for mistakes.
- Focusing on what is best for the project and its users.

Unacceptable behavior includes harassment, personal attacks, public disclosure of private information, and knowingly misleading financial claims.

## Enforcement

Maintainers may remove comments, issues, pull requests, or contributions that violate this code of conduct. Repeated or severe violations may result in a ban from project spaces.
